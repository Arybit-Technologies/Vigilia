# 🛡️ Vigilia – Individual Protection App

**Your Safety, In Your Hands. One App. Total Protection.**

Vigilia is an all-in-one individual protection app that offers 24/7 personal security, emergency alerts, digital privacy tools, health monitoring, and legal support—all from your phone.

> Built with ❤️ using Apache Cordova  
> Developed by: [Arybit Technologies](https://arybit.co.ke)

---

## 🚨 Problem Statement

In today's world, personal safety and digital privacy are increasingly at risk. Vulnerable groups—such as women, children, travelers, and activists—face daily dangers, from physical harm and stalking to data theft and health emergencies. Current solutions are fragmented and inaccessible when needed most.

---

## ✅ Our Solution

**Vigilia** is your personal digital guardian. It seamlessly combines life-saving features into one easy-to-use mobile app that empowers individuals to take control of their safety, health, and digital footprint—anytime, anywhere.

---

## 🌟 Core Features

- 🆘 **Instant SOS Alerts** – One-tap emergency trigger that shares your live location and starts a video/audio stream.
- 📍 **Live Location Tracking** – Real-time GPS tracking with geofencing and safe-zone alerts.
- 🛡️ **Floating Emergency Buttons** – Always-accessible floating action buttons for instant access to emergency features (photo, video, audio, contacts, share location, threat detection, safe route, and refresh location).
- 🔄 **Manual Location Refresh** – Instantly update your GPS location and map with a single tap.
- 🧠 **AI Threat Detection** – Real-time risk analysis using device sensors and AI.
- 🗺️ **Safe Route Planner** – Plan safer travel routes based on real-time data.
- 🌐 **Community Safety Alerts** – Receive and share anonymized alerts about incidents in your area.
- 🎤 **Voice-Activated SOS** – Trigger emergency alerts hands-free with a voice command.
- 🩺 **Health & Medical Monitor** – Vitals monitoring, medication reminders, and health alerts via wearables.
- 🧘 **Mental Health Support** – Access crisis hotlines and mindfulness resources.
- 💬 **Encrypted Chat** – Secure, private communication with trusted contacts.
- 🔐 **Cyber & Privacy Tools** – App permission scanner, unsafe Wi-Fi warnings, and digital hygiene tips.
- 📁 **Evidence Vault** – Securely store photos, videos, and audio as evidence.
- 👩‍⚖️ **Legal ID Vault** – Secure cloud storage for IDs and real-time breach notifications.
- 👥 **Emergency Contacts** – Manage and contact trusted emergency contacts.
- 🧑‍💻 **Multi-Language Support** – Easily switch between supported languages.
- ⌚ **Wearable Device Sync** – Integrate with smartwatches for fall detection and panic button.

## 🛠️ Technical Overview

> For developers interested in contributing or deploying:

**Tech Stack:**

- Framework: Apache Cordova  
- Frontend: HTML5, CSS3, JavaScript  
- Backend: Node.js (planned)  
- Storage: AES-encrypted cloud storage  
- Integrations: GPS, emergency services, wearable APIs  
- Security: AES-256, GDPR/CCPA (planned)

**Basic Setup:**

```bash
# Clone repo
git clone https://github.com/arybit/vigilia.git
cd vigilia

# Install dependencies
npm install

# Add platforms
cordova platform add android
cordova platform add ios

# Build and run
cordova build
cordova run android
```
